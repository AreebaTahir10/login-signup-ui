# Signup & Login Project

A simple, responsive HTML and CSS-based authentication interface with dedicated signup and login pages.

## Project Overview

This project provides a clean and user-friendly interface for user authentication, featuring:
- A **Sign-up page** for new users to create an account
- A **Login page** for existing users to access their account
- Responsive design with modern styling

## Features

✨ **Key Features:**
- Clean and modern UI design
- Fully responsive layout
- Form validation (required fields)
- Centered, card-based design
- Professional styling with shadows and rounded corners
- Light background with white form containers
- Easy navigation between signup and login pages

## Project Structure

```
signup login project/
├── signup.html    # Sign-up page with name, email, and password fields
├── login.html     # Login page with email and password fields
└── style.css      # Shared stylesheet for both pages
```

## Files Description

### `signup.html`
- New user registration form
- Input fields: Name, Email, Password, Confirm Password
- Link to login page for existing users

### `login.html`
- Existing user authentication form
- Input fields: Email, Password
- Simple and straightforward design

### `style.css`
- Unified styling for both pages
- Responsive container layout
- Form styling with labels and inputs
- Flexbox-based centering
- Professional color scheme (light gray background with white containers)

## Getting Started

1. **Open the project**: Open the folder in your preferred code editor
2. **View the pages**: 
   - Open `signup.html` in your browser to view the signup page
   - Open `login.html` in your browser to view the login page
3. **Navigate**: Use the "Login" link on the signup page to switch between pages

## Technologies Used

- **HTML5** - Page structure and form elements
- **CSS3** - Styling and responsive layout
- Flexbox for layout and centering

## Browser Compatibility

Works on all modern browsers that support:
